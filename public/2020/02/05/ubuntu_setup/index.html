Ubuntu Setup

Published Wed, Feb 5, 2020 by Mubasser Kamal

Setting up Ubuntu

  1. Creating a New User 
    adduser mobu
  2. Set up root privileges (sudo) for our normal account 
    usermod -aG sudo mobu

    The command ‘usermod‘ is used to modify or change any attributes of a already created user account via command line.

    -a append the user to the supplemental GROUPS mentioned by the -G option without removing him/her from other groups

    -G new list of supplementary GROUPS

    // to change username, use the -l switch

    usermod -l [old username] [new username]

  3. Enable UFW Firewall & open SSH port 
    // allow port 22 for SSH
sudo ufw allow 22

// enable UFW firewall
ufw enable
  4. Login to new account using SSH 
    rsync --archive --chown=mobu:mobu ~/.ssh /home/mobu

    rsync copies the files, directory structure, and ownership from the .ssh folder of the root to /home/mobu

Securing Ubuntu
